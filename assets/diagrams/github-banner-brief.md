# GitHub Banner Brief

## Goal

Create a clean, modern GitHub social banner for the repository `Python for Data Engineering: Beginner to Advanced`.

## Visual Direction

- professional and technical
- beginner-friendly, not intimidating
- modern data workflow feel
- clear hierarchy
- strong readability at thumbnail size

## Suggested Text

Main title:
`Python for Data Engineering`

Subtitle:
`Beginner to Advanced | Files, APIs, ETL, Databases, Testing, Projects`

## Suggested Prompt

Create a wide GitHub social banner for a technical education repository titled "Python for Data Engineering". The design should feel modern, clean, and professional, with a practical data-engineering atmosphere rather than a generic coding poster. Use a horizontal layout with strong readable typography, subtle data-flow inspired shapes, tables, file icons, API/network motifs, and pipeline-style visual cues. Emphasize clarity, trust, and hands-on learning. Include the exact title text "Python for Data Engineering" and the subtitle "Beginner to Advanced | Files, APIs, ETL, Databases, Testing, Projects". Avoid clutter, avoid cartoonish illustrations, avoid excessive neon effects, avoid watermarks.
