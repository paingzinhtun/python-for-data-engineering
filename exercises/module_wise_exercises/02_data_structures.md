# Module 02 Exercises

## Beginner
1. Create a dictionary for a customer record with `customer_id`, `country`, and `signup_date`.
2. Store three order IDs in a list and print the second one.

## Intermediate
1. Use a set to find duplicate order IDs from a list.
2. Write a function that trims whitespace from every string value in a dictionary.

## Applied
Given a list of order dictionaries, return all unique countries in uppercase sorted order.

## Data Engineering Scenario
API data arrives as nested dictionaries. Extract `customer.id`, `customer.region`, and `order.amount` into a flat dictionary.
