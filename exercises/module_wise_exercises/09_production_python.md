# Module 09 Exercises

## Beginner
1. Add logging statements at script start and finish.
2. Read one environment variable safely.

## Intermediate
1. Write a pytest unit test for a transformation function.
2. Add a CLI argument for input file path.

## Applied
Refactor a single long script into modules for config, transformation, and loading.

## Data Engineering Scenario
Your pipeline fails randomly when reading an API. Decide what should be retried, what should be logged, and what should fail fast.
