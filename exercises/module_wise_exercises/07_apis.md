# Module 07 Exercises

## Beginner
1. Make a GET request with a timeout.
2. Parse a JSON response into a Python object.

## Intermediate
1. Write a function that extracts records from paginated responses.
2. Add a basic retry loop for transient failures.

## Applied
Convert a list of JSON objects from an API into flat dictionaries ready for CSV or database loading.

## Data Engineering Scenario
An API rate-limits requests and sometimes returns missing fields. Design a safe extract strategy.
