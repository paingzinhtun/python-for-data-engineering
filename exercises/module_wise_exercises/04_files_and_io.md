# Module 04 Exercises

## Beginner
1. Read a text file line by line using `pathlib`.
2. Write a JSON file with two records.

## Intermediate
1. Read a CSV file using `csv.DictReader`.
2. List all `.csv` files inside a folder.

## Applied
Write a file validation function that checks existence, suffix, and minimum size.

## Data Engineering Scenario
You receive a daily drop folder with mixed files. Write the steps or code outline to move valid CSV files to processing and invalid files to quarantine.
