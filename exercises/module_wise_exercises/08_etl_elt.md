# Module 08 Exercises

## Beginner
1. Write down the difference between ETL and ELT.
2. Define raw, processed, and curated layers in your own words.

## Intermediate
1. Split an existing script into extract, transform, and load functions.
2. Add a run summary with row counts.

## Applied
Design a config-driven batch pipeline that reads one input file, transforms records, and loads SQLite output.

## Data Engineering Scenario
Your pipeline is rerun for the same date. Describe how you would make the process idempotent.
