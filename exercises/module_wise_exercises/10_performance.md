# Module 10 Exercises

## Beginner
1. Explain the difference between a list and a generator.
2. Give one example of a memory-heavy pattern.

## Intermediate
1. Rewrite a file-reading function to yield rows one at a time.
2. Describe when chunked processing helps.

## Applied
Estimate the tradeoffs between processing 10,000 rows in pure Python and processing 10 million rows with a single-machine script.

## Data Engineering Scenario
Your current script reads an entire multi-GB file into memory and crashes. Propose a safer design.
