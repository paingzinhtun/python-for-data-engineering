# Module 01 Exercises

## Beginner
1. Create variables for a source file name, row count, and whether the file is valid.
2. Write a function called `is_large_file` that returns `True` when a file has more than 1000 rows.

## Intermediate
1. Loop through a list of file names and print only the `.csv` files.
2. Write a function that returns `"keep"` when a row has a positive amount and `"reject"` otherwise.

## Applied
Write a script that counts how many file names in a list look like daily sales extracts, for example `sales_2026_03_29.csv`.

## Data Engineering Scenario
You receive a list of file metadata records with `name`, `size_bytes`, and `extension`. Write logic to decide which files are safe to process.
