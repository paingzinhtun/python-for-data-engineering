# Module 03 Exercises

## Beginner
1. Write a list comprehension that keeps only active records.
2. Write a function with a default argument called `parse_float`.

## Intermediate
1. Create a module-level helper function and describe when it should live in a separate file.
2. Use `try` and `except` to handle invalid integer conversion.

## Applied
Sort a list of dictionaries by `created_at` descending and then by `customer_id`.

## Data Engineering Scenario
Design a small `FileJob` class or a plain-function alternative that tracks a file path and whether processing succeeded. Explain which design you chose and why.
