# Module 05 Exercises

## Beginner
1. Convert a string amount into a float.
2. Replace blank country values with `"UNKNOWN"`.

## Intermediate
1. Write a function that validates required fields.
2. Aggregate total sales amount by country using basic Python only.

## Applied
Normalize a list of order records into a consistent schema with proper field names and types.

## Data Engineering Scenario
You receive messy sales data with missing IDs, negative quantities, and mixed-case country codes. Design a transformation pipeline that separates valid and invalid rows.
