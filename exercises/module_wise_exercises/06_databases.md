# Module 06 Exercises

## Beginner
1. Connect to SQLite and create a table.
2. Insert one record with a parameterized query.

## Intermediate
1. Insert many records with `executemany`.
2. Query all rows with amount greater than 100.

## Applied
Load transformed order data into a table with a transaction and rollback on failure.

## Data Engineering Scenario
Design a loading function that prevents duplicate inserts when the same batch is rerun.
