# Repo Navigation Guide

## Start Here

- `README.md`
- `docs/course-overview.md`
- `docs/setup-guide.md`
- `modules/00_orientation/README.md`

## Where To Learn

- `modules/` contains the main lessons
- `exercises/module_wise_exercises/` contains practice work
- `exercises/solutions/` contains solutions after you try the exercises

## Where To Build

- `src/python_for_data_engineering/` contains reusable code
- `projects/mini_projects/` contains smaller guided projects
- `projects/capstones/` contains larger end-to-end projects

## Where To Test

- `tests/unit/` contains fast function-level tests
- `tests/integration/` contains pipeline-level tests
