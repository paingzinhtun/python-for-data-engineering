# Learning Path

## Recommended Sequence

If you are completely new to Python:

1. Module 00: Orientation
2. Module 01: Python Basics
3. Module 02: Data Structures
4. Module 03: Intermediate Python
5. Module 04: Files and I/O
6. Module 05: Data Processing
7. Module 06: Databases
8. Module 07: APIs
9. Module 08: ETL / ELT
10. Module 09: Production Python
11. Module 10: Performance
12. Module 11: Project Structure
13. Module 12: Real-World Patterns
14. Module 13: Mini Projects
15. Module 14: Capstones
16. Module 15: Interview Prep
17. Module 16: Next Steps

## If You Already Know Basic Python

- Skim modules 01 and 02
- Spend time on modules 04 to 12
- Complete at least one database project and one API project
- Use module 15 to review interview-style reasoning

## How To Pace Yourself

### Light pace
- 3 to 5 hours per week
- Finish in 10 to 12 weeks

### Standard pace
- 5 to 8 hours per week
- Finish in 6 to 8 weeks

### Intensive pace
- 8 to 12 hours per week
- Finish in 4 to 5 weeks

## Checkpoints

### Checkpoint 1
After module 03, you should be able to:

- write functions
- work with lists and dictionaries
- handle simple exceptions
- organize small scripts into modules

### Checkpoint 2
After module 06, you should be able to:

- read CSV and JSON files
- clean raw records
- validate fields
- load data into SQLite

### Checkpoint 3
After module 09, you should be able to:

- add logging
- use config files and environment variables
- write tests for transformation logic
- structure a simple pipeline

### Checkpoint 4
After module 14, you should be able to:

- explain a pipeline architecture
- build a portfolio-ready project
- reason about batch workflow tradeoffs
- discuss common failure scenarios
