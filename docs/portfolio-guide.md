# Portfolio Guide

## Turning Lessons Into Portfolio Work

Employers care less about whether you completed a lesson and more about whether you can show engineering judgment.

## What To Showcase

- a clear project README
- a realistic problem statement
- sample input data
- pipeline steps and architecture
- logging and validation
- tests for key functions
- tradeoff notes and future improvements

## Strong Project Signals

- you separated raw and processed data
- you documented schema assumptions
- you handled bad records
- you used parameterized queries
- you thought about reruns and duplicate loads
- you explained when the design would stop scaling
